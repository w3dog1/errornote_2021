# errornote_2021
django errornote project

## Intro
- Errornote는 개인의 Error issue들을 기록하고, 다 함께 공유하기 위해 만들어진 웹 서비스입니다!

## Features
- **Django 3.0ver & Python 2.7**
- Install via **pip3**
- Server with **AWS**
- User log in/out, sign up, password via **Django-allauth**
- Styling with **Bootstrap v5**


## Installation
Clone this repo, install Django
> $ git clone https://github.com/w3dog1/errornote_2021.git
> <br>
> $ cd errornote_2021
> 
> $ pip3 install django
> 
> $ python3 manage.py migrate
> <br>
> $ python3 manage.py runserver

Find the app running at http://localhost:8000/
